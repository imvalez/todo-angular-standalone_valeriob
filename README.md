# Todo Angular Standalone

Todo Angular Standalone is a portfolio project that demonstrates Angular standalone components in a practical task-tracking app.

## Tech stack

- Angular 19
- TypeScript
- Express SSR adapter (generated by Angular)
- LocalStorage persistence

## Features

- add, complete, and delete todos
- search and status filtering (all, active, completed)
- clear completed tasks
- persistent state in browser localStorage

